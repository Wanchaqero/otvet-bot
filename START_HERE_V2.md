# 🚀 НАЧНИТЕ ОТСЮДА v2.0 - УМНЫЙ БОТ

## 🧠 ЧТО НОВОГО?

Теперь у вас есть **два варианта** бота:

### 1️⃣ smart-bot.js (РЕКОМЕНДУЕТСЯ) 🤖⭐
```
Пользователь: "Нас 2 чел, 12 дней. Туры в Лиму, Паракас, Кузко"
    ↓
Claude AI анализирует и находит все подходящие туры
    ↓
Бот отправляет красивый ответ с рекомендациями
```
**Требует:** ANTHROPIC_API_KEY (Claude API)
**Стоимость:** ~$1-10/месяц

### 2️⃣ bot.js (БАЗОВЫЙ)
```
Пользователь: "туры по Лиме"
    ↓
Бот ищет "Лима" в БД
    ↓
Отправляет результаты
```
**Требует:** Только TELEGRAM_BOT_TOKEN
**Стоимость:** $0

## ✅ БЫСТРАЯ УСТАНОВКА

### Шаг 1: Получить ключи

**Telegram токен:**
- @BotFather → `/newbot` → скопировать токен

**Claude API ключ (опционально, для умного бота):**
- https://console.anthropic.com → "API keys" → "Create Key"

### Шаг 2: Добавить ключи в .env

```bash
TELEGRAM_BOT_TOKEN=123456:ABCdef...
ANTHROPIC_API_KEY=sk-ant-xxxxx...  # если хотите умный бот
ALLOWED_USERS=7575536082,123456789
```

### Шаг 3: На GitHub

```bash
git init
git add .
git commit -m "BerúPeru Smart Bot"
git branch -M main
git remote add origin https://github.com/ВАШ_USERNAME/beruperu-bot.git
git push -u origin main
```

### Шаг 4: На Railway

1. railway.app → "New Project" → "Deploy from GitHub"
2. Выберите `beruperu-bot`
3. Добавьте переменные:
   - `TELEGRAM_BOT_TOKEN`
   - `ALLOWED_USERS`
   - `ANTHROPIC_API_KEY` (если нужен умный бот)
4. Save → Railway перезагрузится!
