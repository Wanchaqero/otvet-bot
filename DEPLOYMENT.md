## 🚀 Шаг 1: GitHub

### Создайте репозиторий

1. На github.com нажмите **"+"** → **"New repository"**
2. Назовите: `beruperu-bot`
3. Выберите **"Public"**
4. Нажмите **"Create repository"**

### Загрузьте файлы

**Вариант 1: Через веб (проще)**

1. В репо нажмите **"Add file"** → **"Upload files"**
2. Выберите все файлы из папки `F:\Work\SiteBD` (кроме `node_modules/`)
3. Нажмите **"Commit changes"**

**Вариант 2: Через Git**

```bash
cd F:\Work\SiteBD
git init
git add .
git commit -m "Initial commit - BerúPeru Bot"
git branch -M main
git remote add origin https://github.com/ВАШ_USERNAME/beruperu-bot.git
git push -u origin main
```

## 🚀 Шаг 2: Railway.app

### Создайте проект

1. На railway.app нажмите **"New Project"**
2. Выберите **"Deploy from GitHub"**
3. Выберите репозиторий `beruperu-bot`

Railway автоматически:
- ✅ Обнаружит Node.js проект
- ✅ Установит зависимости
- ✅ Запустит бота

### Добавьте переменные окружения

1. Нажмите на проект → **"Variables"**
2. Нажмите **"+ New Variable"** для каждой:

| Переменная | Значение |
|-----------|----------|
| `TELEGRAM_BOT_TOKEN` | Токен от @BotFather |
| `ALLOWED_USERS` | `7575536082,123456789` (ваши ID) |

3. Нажмите **"Save"**

Railway автоматически перезагрузится с новыми переменными!

---

## 🔑 Получить токен бота

1. Откройте Telegram
2. Найдите **@BotFather**
3. Отправьте `/newbot`
4. Следуйте инструкциям
5. Скопируйте токен (выглядит так: `123456789:ABCdefGHIjklmnoPQRstuvWXYZ`)

---

## 👤 Получить свой ID в Telegram

1. Найдите **@userinfobot**
2. Нажмите **"Start"**
3. Скопируйте ваш ID
4. Добавьте в переменную `ALLOWED_USERS`

## 🔄 Обновление туров

### Вариант 1: Закинуть новую tours.db

1. Локально спарсьте сайт: `npm run parse`
2. Добавьте `tours.db` в git
3. Сделайте `git push`
4. Railway обновит бота

### Вариант 2: Спарсить на Railway

1. В Railway откройте **"Console"**
2. Выполните: `npm run parse`
3. Все готово!

## 🆘 Если что-то не работает

### ❌ "Cannot find module 'sqlite3'"

Railway устанавливает зависимости автоматически. Проверьте логи.

### ❌ "U вас нет доступа к этому боту"

Проверьте переменную `ALLOWED_USERS` - добавьте свой ID.

### ❌ Бот не отвечает

1. Проверьте что в логах нет ошибок
2. Убедитесь что токен правильный
3. Перезагрузите Railway
