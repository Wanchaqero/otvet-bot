# 🌍 BerúPeru AI Telegram Bot v2.0

Telegram бот для туристического агентства с базой данных SQLite и проверкой прав доступа.

## ✨ Возможности

- 🔍 **Интеллектуальный поиск** туров по БД (супер быстро!)
- 🔒 **Проверка прав доступа** - только разрешенные пользователи могут писать
- 📊 **SQLite БД** - хранит туры, статистику, избранное
- 💬 **Натуральный язык** - понимает запросы на русском
- 📱 **Telegram интеграция** - удобное общение через мессенджер
- 🔗 **Прямые ссылки** - клики на туры сразу с результатов

## 🚀 Быстрый старт

### 1. Установить зависимости

```bash
npm install
```

### 2. Спарсить сайт ОДИН РАЗ

```bash
npm run parse
```

Это создаст файл `tours.db` с вашими турами из сайта.

### 3. Добавить токен в .env

```bash
# Отредактируйте .env файл:
TELEGRAM_BOT_TOKEN=ваш_токен_от_бот_фатера
ALLOWED_USERS=7575536082,123456789  # ваши ID в Telegram
```

### 4. Запустить бота

```bash
npm start
```

**Готово!** Бот работает! 🎉

## 📁 Файлы проекта

- **bot.js** - Основной Telegram бот с проверкой прав
- **database.js** - Класс для работы с SQLite БД
- **parse-and-save.js** - Скрипт парсинга сайта
- **schema.sql** - Структура БД
- **package.json** - Зависимости
- **.env** - Конфигурация (токен, разрешенные пользователи)

## 🔒 Проверка прав доступа

Добавьте разрешенные ID пользователей в **.env**:

```bash
ALLOWED_USERS=7575536082,123456789,987654321
```

Или переменную окружения на Railway.

## 📊 Как работает

1. **Первый раз:** `npm run parse` парсит сайт и сохраняет туры в `tours.db`
2. **Всегда:** бот ищет туры в БД (очень быстро!)
3. **Результаты:** отправляет в Telegram со ссылками на полное описание

## 🌐 Развертывание на Railway.app

1. Залейте этот репо на GitHub
2. На Railway.app создайте новый проект из GitHub
3. Добавьте переменные окружения:
   - `TELEGRAM_BOT_TOKEN` - ваш токен
   - `ALLOWED_USERS` - разрешенные ID (через запятую)
4. Railway автоматически запустит бота

## 🔄 Обновление туров

Если туры изменились на сайте:

```bash
npm run parse
```

Это перепарсит сайт и обновит tours.db.
