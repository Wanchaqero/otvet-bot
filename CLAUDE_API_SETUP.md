## 🔑 Получить Claude API ключ

### Шаг 1: Создать аккаунт

1. Откройте [console.anthropic.com](https://console.anthropic.com)
2. Нажмите **"Sign up"** или **"Log in"**
3. Создайте аккаунт (используйте реальный email)

### Шаг 2: Создать API ключ

1. В левом меню нажмите **"API keys"**
2. Нажмите **"Create Key"**
3. Дайте имя (например: "BerúPeru Bot")
4. Скопируйте ключ (выглядит так: `sk-ant-xxxxxxxxxxxxxxxxxxxxx`)

### Шаг 3: Добавить в .env локально

```bash
# Отредактируйте .env:
ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxxxxxxxxxx
```

### Шаг 4: На Railway добавить переменную

1. Railway.app → ваш проект
2. **Variables** → **+ New Variable**
3. KEY: `ANTHROPIC_API_KEY`
4. VALUE: `sk-ant-xxxxxxxxxxxxxxxxxxxxx`
5. Save

## 💰 Стоимость

### Модель Claude 3.5 Sonnet

- **Input**: $3 за 1M токенов
- **Output**: $15 за 1M токенов

### Примерный расход на анализ

1 сложный запрос ≈ 500 входящих + 300 исходящих токенов
= ~0.003¢ за запрос

**На месяц:**
- 100 сложных запросов в день = ~$10/месяц
- 10 запросов в день = ~$1/месяц

Очень дешево! 💸

## 🤖 Как работает умный бот

### Поток данных

```
Пользователь отправляет сложный запрос
         ↓
Claude анализирует:
  - Какие локации упомянуты?
  - Сколько людей?
  - Какой бюджет?
  - Сколько дней?
  - Какие интересы?
         ↓
Бот ищет в БД по найденным ключевым словам
         ↓
Claude генерирует красивый ответ:
  - Подтверждает понимание запроса
  - Объясняет почему эти туры подходят
  - Предлагает синтез маршрута
         ↓
Отправляем с ссылками на туры в Telegram
```

### Пример анализа

Запрос:
```
"Здравствуйте. Поездку по Перу планируем на начало сентября. 
Нас 2 чел. Отпуск 12 дней. Туры по Лиме, Паракасе, 
Куско, Палкойо, мост Кесвачака"
```

Claude извлекает:
```json
{
  "locations": ["Лима", "Паракас", "Кузко", "Палкойо"],
  "keywords": ["музей", "остров", "долина", "гора", "мост"],
  "group_size": "2 человека",
  "dates": "начало сентября",
  "duration": "12 дней",
  "summary": "Туристы ищут разнообразный маршрут..."
}
```

Бот ищет в БД и находит 6-7 подходящих туров! ✅

|-----------|----------|

## 🐛 Если что-то не работает

### ❌ "ANTHROPIC_API_KEY не установлен"

Бот переходит в базовый режим без ИИ. Это нормально!

```
⚠️ ANTHROPIC_API_KEY не установлена. 
Бот будет работать в базовом режиме (без ИИ).
```

### ❌ "Invalid API key"

Проверьте что ключ скопирован без пробелов:
- Правильно: `sk-ant-xxxxx`
- Неправильно: ` sk-ant-xxxxx ` (пробелы)

### ❌ "Claude не отвечает"

Может быть перегруз или проблема с сервером Claude. Бот автоматически вернется к базовому поиску.
